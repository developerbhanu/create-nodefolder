# create-nodefolder

Nodefolder is short for nodejs scaffolder, that will scaffold a generic nodejs project so that you don't have to do most of the configuration or setup.

## How to generate:

1. use ` npm create nodefolder ` to create a new project.

2. navigate to the created project: `cd <project-name>`.

2. copy `.env.example` and create `.env` file and assign values as required.

3.  run `npm install` to install dependencies.

4. run `npm run start` to spin up the server.
