# create-nodefolder

Nodefolder is short for nodejs scaffolder, that will scaffold a generic nodejs project so that you don't have to do most of the configuration or setup.

## How to generate:

1. Navigate to your desired directory and type:

``` npm create nodefolder ```

2. copy `.env.example` and create `.env` file and make changes as required.

3.  run `npm install` to install dependencies.

4. run `npm run start` to spin up the server.
